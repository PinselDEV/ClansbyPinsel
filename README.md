# ClansbyPinsel

## ğŸ“œ Commands

### Player Commands

| Command | Description |
|--------|-------------|
| `/clan` | Main command / help |
| `/clan create <name>` | Create a clan |
| `/clan invite <player>` | Invite a player |
| `/clan accept` | Accept invite |
| `/clan deny` | Deny invite |
| `/clan leave` | Leave clan |
| `/clan chat` | Toggle clan chat |
| `/clan home` | Teleport to clan home |
| `/clan sethome` | Set clan home |
| `/clan info` | Show clan info |
| `/clan kick <player>` | Kick member |
| `/clan promote <player>` | Promote member |
| `/clan demote <player>` | Demote member |
| `/clan disband` | Delete clan |
| `/clan gui` | Open clan GUI |

---

### Admin Commands

| Command | Description |
|--------|-------------|
| `/clanadmin reload` | Reload plugin |
| `/clanadmin disband <clan>` | Delete a clan |
| `/clanadmin setowner <clan> <player>` | Set clan leader |
| `/clanadmin add <clan> <player>` | Add player to clan |
| `/clanadmin remove <player>` | Remove player from clan |
| `/clanadmin info <clan>` | View clan info |
